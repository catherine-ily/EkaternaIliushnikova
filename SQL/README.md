# SQL-проекты

## О чем эта папка

Здесь собраны учебные проекты по **PostgreSQL** и работе с данными. Основной фокус — написание аналитических запросов, ETL-обработка и оптимизация.

## Как запустить
  
1. Установите PostgreSQL.
2. Для запуска проекта необходимо скачать скрипт и применить его к базе данных. Для всех файлов кроме: `SQL6_salesman_problems`, `SQL12_currency_calculation_1`, `SQL12_currency_calculation_2` актуален скрипт по пути: *materials/Pizzeria/model*, в той же папке хранится визуализация модели базы данных, с которой мы работаем.
3. Для запуска файла `SQL6_salesman_problems` достаточно пустой базы данных - таблицы в данной проекте создаются и заполняются с помощью скрипта в самом файле.
4. Для запуска проектов по работе с валютой - `SQL12_currency_calculation_1` и `SQL12_currency_calculation_2` - потребуется предварительно скачать и применить к базе данных скрипт по пути: *materials/WarehousFundamentals/model*, в той же папке хранится визуализация модели базы данных, с которой мы работаем.
5. Для проверки результатов используйте `psql` или любой клиент (PGAdmin, DataGrip).
