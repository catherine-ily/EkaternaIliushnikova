# Проекты по бизнес-анализу (BSA)

## О чем эта папка

Здесь собраны учебные проекты по **бизнес-анализу** и моделированию требований. Основной фокус — визуализация процессов, формализация требований и работа с документацией.

Данная директория содержит две папки: *Barbershop* и *Delivery*. В них собраны файлы с реализацией проектов по каждой из двух задач: создание системы записи на услуги барбершопа и системы стартапа по доставке заказов.

Стоит обратить внимание на то, что в соответствии с заданиями проектов, не для обеих задач выполнялся одинаковый набор действий: одна из задач обычно выступала примером, такие файлы в репозиторий я вносить не стала - здесь находятся только файлы, выполненные мной.

Так, например, для задачи *Barbershop* были составлены BPMN-диаграммы бизнес-процессов, а для задачи *Delivery* - задокументирован процесс выявления бизнес-требований.

---


## Что я умею (по итогам этих проектов)

- Выявлять список заинтересованных лиц и их потребностей
- Выявлять и документировать бизнес-требования
- Строить DFD, Swimlane, BPMN, диаграммы состояний
- Сравнивать разные модели требований, выявлять и исправлять нестыковки
- Писать User Stories с критериями приёмки и тестовыми сценариями
- Составлять Use Case (варианты использования) и сценарии взаимодействия
- Проводить командные обсуждения требований
- Строить логическую модель базы данных и словарь данных
