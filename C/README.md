# C-проекты

## О чем эта папка

Здесь собраны учебные проекты на языке **C**. Они демонстрируют алгоритмическое мышление, работу с памятью и реализацию базовых библиотек с нуля.

В каждой папке сохранены некоторые, важные на мой взгляд, директории из учебных проектов: примеры кода, данных, материалы и файлы README с текстом заданий. Сами файлы проектов хранятся в папках `src`.

## Как собрать и запустить

### Требования
- Unix-подобная операционная система (Linux, macOS, WSL2 on Windows)
- Компилятор GCC, Clang
- Make
- Библиотека Check для запуска тестов

### Запуск проекта C1_SimpleBashUtils
- Зайти в директорию с проектом (папка /src/cat или /src/grep)
- В терминале запустить команду Make - произойдет сборка объектного файла (с расширением .o)
- Для запуска теста выполнить команду: *./test_for_cat* или *./test_for_grep* для проверки утилиты `cat` или `grep` соответственно.

### Запуск для проектов с реализацией библиотек (C2_stringplus, C3_decimal, C4_matrix)
- Зайти в директорию с проектом (папка /src)
- В терминале запустить команду Make - произойдет сборка библиотеки (файл с расширением .а)
- Makefile поддерживает запуск команд: 
    - clean: удаление объектных файлов и отчета
    - test: запуск unit-тестов
    - gcov_report: формирование отчета о покрытии кода тестами
    - clang: проверяет код на соответствие стилю
